# The Orchestra of Bandit Wolves — Website

This repository contains the source code for **The Orchestra of Bandit Wolves**, a cinematic music portfolio and storefront by media composer Kelvin Chico.

The site serves mindful explorers, filmmakers, and creative professionals by presenting curated music, licensing materials, written chronicles, and digital products in a refined, minimal web experience.

---

## Overview

The website functions as:
- A composer portfolio and listening space
- A sync-licensing gateway for visual media
- A storefront for campaign bundles, sample packs, and digital releases
- A central hub for newsletters and long-form creative writing

The site is intentionally lightweight, static, and fast—designed for clarity, longevity, and ease of maintenance.

---

## Tech Stack

- **HTML5** — semantic, accessible markup  
- **CSS3** — custom properties, responsive layout  
- **Vanilla JavaScript** — minimal interactivity only  
- **GitHub** — version control  
- **Netlify** — deployment, hosting, and SSL  
- **Substack** — newsletter delivery  
- **Disco** — streaming and download access for music  

No build tools or frameworks are required.
